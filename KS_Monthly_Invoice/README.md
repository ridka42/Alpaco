# KS_Monthly_Invoice
> ACME에 업로드 된 월별 인보이스 엑셀 파일을 다운로드하여 정합성 검증을 수행하는 프로세스


월별 인보이스 엑셀 파일을 수기로 검토 가능하도록 월별 폴더로 보존한다.
임의의 특정 월에 대해서만 검증 수행될 수 있도록 분기처리 한다.
RPA 수행한 결과에 따라 각각 분기 처리하여 이메일을 전송한다.
